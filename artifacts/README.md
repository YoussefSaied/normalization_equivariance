# Artifact Layout

- `paper/`: paper PDFs linked from the README.
- `figures/denoising/`: README-facing denoising and mismatch figures.
- `figures/mechanism/`: README-facing mechanism and analysis figures.
- `figures/sampler/`: README-facing sampler trajectories.
- `analysis/`: detailed analysis outputs used to build paper figures.
- `speed_tests/`: raw runtime benchmark outputs.
